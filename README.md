Task 2 – Dynamic To-Do List Using JavaScript
Project Title
Dynamic To-Do List Web Application

Description

This project is a dynamic To-Do List web application developed using HTML, CSS, and JavaScript. The application allows users to add and remove tasks dynamically using JavaScript DOM manipulation. This project helps in understanding how JavaScript interacts with HTML elements in real time to build interactive web applications.

Objective

To learn and apply DOM manipulation concepts in JavaScript by creating a functional and interactive to-do list.

Technologies Used

HTML5

CSS3

JavaScript (DOM Manipulation)

Features

Add new tasks dynamically

Remove tasks from the list

Real-time UI updates without page reload

Simple and user-friendly interface

Responsive design

Project Structure
/task-2-todo-list
│
├── index.html
├── style.css
└── script.js

How It Works

Users enter a task in the input field. When the Add Task button is clicked, JavaScript creates a new list item using DOM methods. Each task includes a delete button to remove the task dynamically. All updates happen instantly without refreshing the page.

Concepts Covered

DOM selection and manipulation

Event listeners

Creating and removing HTML elements using JavaScript

Basic UI interaction

JavaScript functions

Output

A fully functional dynamic to-do list that allows users to manage tasks by adding and deleting them easily.
